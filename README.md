# Alan Labaš — Portfolio

Personal portfolio website built with React + Vite.

## Live
[Alan Labas](https://alan-labas.site)

##  Built with
- React + Vite
- CSS Variables (dark/light mode)
- React Context (theme + language)
- Deployed on Vercel

##  Features
- Dark / Light mode
- Bilingual (EN / SL)
- Responsive design
- Subtle scroll animations

##  Run locally
- git clone https://github.com/Alan-Labas/portfolio
- cd portfolio
- npm install
- npm run dev
